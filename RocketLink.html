RocketLink

Cloud - File Sharing Platform

This project was made during the Web & Mobile Development course.

The goal was to develop a Cloud platform, that allowed to share files by sending a link(url). What I was able to develop in this project was similar to WE TRANSFER.

For this project I used the following technologies:

- JavaScript
- Php
- HTML & CSS
- Bootstrap
- Windows Task Scheduler (Command Prompt)
- XAMPP
